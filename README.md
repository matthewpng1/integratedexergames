Integrated Exergames

The Unity game source code used for a final year project to encourage elderly users to exercise their hip and grip

Features
ESP32 Bluetooth Controller: Cost-efficient (<$100) hardware built with off-the-shelf components for accessibility and rapid prototyping
Unity Game Integration: Interactive exergame designed to encourage physical activity, featuring engaging game mechanics tailored for older adults
User-Centered Design: Conducted structured user testing, achieving a mean PACES score of 44.1/56, demonstrating high user enjoyment and engagement
Affordable & Deployable: Designed with scalability and ease of use in mind, emphasizing practical deployment in community settings
Purpose

Designed to address sedentary behavior among elderly populations by providing an engaging, low-cost exercise solution. The project combines hardware design, software development, and user evaluation to create a functional, real-world system.

Tech Stack
Hardware: ESP32, C++ programming, Bluetooth HID
Software: Unity, C#
Data Analysis: PACES questionnaire for quantitative assessment
Prototyping: Off-the-shelf components, 3D printed parts for controller housing
Getting Started
Clone the repository
Open the Unity project for the game component
Follow instructions in the hardware folder to build the ESP32 controller
Run the system to connect controller input to the Unity game
Results
Structured user testing confirmed usability and engagement
Demonstrated the effectiveness of a low-cost, integrated exergame system for older adults
